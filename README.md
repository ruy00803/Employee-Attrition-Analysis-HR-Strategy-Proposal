# Employee Attrition Analysis & HR Reform Strategy


架空のIT企業における「業界水準を上回る離職率（16.19%）」という経営課題に対し、データ主導で真因を特定し、改善策を提案したケーススタディです。東京大学 GCI（グローバル消費インテリジェンス寄付講座）の最終課題として作成しました。

## 💡 Key Insights 


現状のデータ分析から、離職を引き起こす2つの「評価制度の歪み」が明らかになりました。

成果と評価が連動していない
達成額に大きな差（0-5kドル〜20k+ドル）があるにもかかわらず、平均パフォーマンス評価は「2.444〜2.671」とほぼ横ばいであり、成果が報われない構造になっています。

評価基準が形骸化している
人事による評価の過半数（52.5%）が「Excellent」以上となっており、評価の差がつきにくくなっています。

これらの結果、「最も会社に貢献しているハイパフォーマーが最も報われず、意欲を失い離職していく」という構造的な問題が発生しています。一般的にハイパフォーマー1名の離職コストは年収の1.5倍〜2倍（年収600万なら約900万円の損失）と試算され、経営に大きなインパクトを与えます。

## 🛠️ 技術的な取り組み (機械学習モデルによる検証)
離職リスクを算出・要因特定するために、機械学習モデルを構築しました。

使用アルゴリズム: LightGBM (勾配ブースティング決定木)

複雑な変数間の関係を捉える精度が高く、どの変数が重要か（変数重要度）を可視化できるため採用。

精度評価: AUC 0.8486

「離職しやすい社員」を高い精度で特定可能であることを示しています。

モデルからの示唆（変数重要度）:

離職予測に圧倒的に最も影響を与えた変数は MonthlyIncome（月給） と Incentive（インセンティブ） でした。報酬額が社員の定着を左右する最大の要因であることが裏付けられました。

次いで「勤続年数」や「年齢」が上位に入り、経験に見合った報酬が得られていない不満が示唆されます。

一方、「パフォーマンス評価」の影響度はTop10圏外と低く、評価制度が社員の意思決定において重要視されていない（機能不全である）ことが確認されました。

## 🚀 解決策の提案とシミュレーション
データ分析の結果に基づき、「成果に正しく報いる」インセンティブ施策を提案し、その効果をモデルを用いてシミュレーションしました。

仮説検証シミュレーション
「高実績者の月給・インセンティブを20%向上させた場合」を仮定し、モデルに入力して離職リスクを再計算しました。

結果: 対象者の離職リスクが 1.09% 低下 すると予測されました。

懸念点の払拭: 収入増によるプレッシャー増大（ストレス評価の悪化）を懸念しシミュレーションした結果、予測されるストレス評価の変化はわずか「+0.008%」でした。本施策はウェルビーイングを損なうことなく、安全に実行可能です。
